<span data-ttu-id="89927-104">Aðgerðin **Enduráætla** er venjulega notuð eftir íhlutum, sem standa fyrir undirliggjandi framleiðslupantanir, hefur verið bætt við eða þeim breytt.</span><span class="sxs-lookup"><span data-stu-id="89927-104">The **Replan** function on production orders is typically used after you have added or changed components that constitute underlying production orders.</span></span> <span data-ttu-id="89927-105">Þessi áætlunaraðgerð reiknar breytingar sem gerðar eru á íhlutum og leiðarlínum. <span data-ttu-id="89927-106">Aðgerðin Enduráætla byggir útreikninga og áætlun á nýrri eftirspurn fyrir framleiðslupöntun á breytingum sem gerðar hafa verið á íhlutum og leiðarlínum.</span><span class="sxs-lookup"><span data-stu-id="89927-106">Based on the changes you have made to the components and routing lines, the Replan function calculates and plans for any new demand for the production order.</span></span>

<span data-ttu-id="89927-110">Uppfæra virknin reiknar breytingar sem gerðar eru á framleiðslupantanahaus og tekur ekki með framleiðsluuppskriftarstig.</span><span class="sxs-lookup"><span data-stu-id="89927-110">The Refresh function calculates changes made to a production order header and does not involve production BOM levels.</span></span> <span data-ttu-id="89927-111">Aðgerðin reiknar og virkjar gildi íhluta- og leiðarlínanna í samræmi við aðalgögnin, sem skilgreind eru í úthlutaðri framleiðsluuppskrift og leið, eftir pantanamagni og skiladegi í haus framleiðslupöntunar.</span><span class="sxs-lookup"><span data-stu-id="89927-111">The function calculates and initiates the values of the component lines and routing lines based on the master data defined in the assigned production BOM and routing, according to the order quantity and due date on the production order’s header.</span></span>

<span data-ttu-id="89927-112">Hægt er að setja framleiðslulínur inn handvirkt eða nota aðgerðina sem reiknar framleiðslupöntunarlínurnar úr hausnum.</span><span class="sxs-lookup"><span data-stu-id="89927-112">You can either insert the production order lines manually or use the function that calculates the production order lines from the header.</span></span>  

> [!NOTE]
 <span data-ttu-id="89927-113">Ef aðgerðin Uppfæra er notuð til að endurreikna framleiðslupöntunarlínur, er eldri framleiðslupöntunarlínunum eytt og nýjar línur eru reiknaðar.</span><span class="sxs-lookup"><span data-stu-id="89927-113">If you use the Refresh function to recalculate production order lines, the old production order lines are deleted and new lines are calculated.</span></span>
